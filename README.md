# Trade Opportunities API

## 📌 Overview
This project is a FastAPI-based backend service that analyzes Indian market sectors and provides structured trade opportunity insights using AI.

The API collects sector-related data, processes it using Google's Gemini LLM, and returns a professional markdown report.

---

## 🚀 Features

- 🔍 Sector-based market analysis
- 🤖 AI-powered insights using Gemini API
- 📊 Structured markdown reports
- 🔐 JWT Authentication
- ⚡ Rate limiting (5 requests/minute)
- 🧠 Session tracking (in-memory)
- ✅ Input validation & error handling

---

## 🛠 Tech Stack

- FastAPI
- Python (Async)
- Google Gemini API
- SlowAPI (Rate Limiting)
- JWT Authentication
